\# 勤怠管理システム



\## 概要

従業員の勤怠・交通費を管理するWebアプリケーションです。



\## 背景

業務で使われている勤怠システムを参考に、実務を意識して開発しました。



\## 機能一覧

\- ログイン機能

\- ユーザー登録

\- 勤怠入力（出退勤・休憩）

\- 勤怠自動計算（残業・深夜・休日）

\- 交通費申請

\- 権限管理（管理者・一般ユーザー）



\## 使用技術

\- Java（Spring Boot）

\- MyBatis

\- HTML / CSS / JavaScript

\- H2（開発用） / PostgreSQL（予定）



\## 工夫した点

\- 勤怠時間の自動計算ロジックを実装

\- バリデーションのグループ分けによる条件分岐

\- 管理者と一般ユーザーの権限分離
