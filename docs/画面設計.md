# 画面遷移図

## ■概要

本システムは以下の画面で構成される。

- ログイン画面
- ユーザー一覧画面（管理者）
- 勤怠入力画面
- 勤怠一覧画面
- 交通費・経費入力画面

---

## ■遷移の特徴

- ロールに応じて遷移を制御（管理者 / 一般ユーザー）
- 勤怠・交通費・経費はそれぞれ申請フローあり
- 一覧画面から詳細・入力画面へ遷移可能
